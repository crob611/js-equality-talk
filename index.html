Strict Equality


              '1' === 1  // false
            

              0 === false  // false

Loose Equality


              '1' == 1  // true
            

              0 == false  // true
  • strict equality, are they exactly equal
  • loose equality, is it close enough
  • loose trys to get them to the same type if possible, then does strict
  • getting them to the same type is type coercion

Summary

  • Primitive things are passed around by value
  • Objects (or arrays, or functions) are passed around by reference
  • Value types are equal if they have the same value
  • Reference types are equal if they reference the same thing
